navbar\n         ↓\n   Show count (e.g., \"Notifications (5)\")\n```\n\n**Real-time:** Updates within <1 second  \n**No UI reload needed:** Badge updates instantly  \n**PWA support:** Updates OS app icon badge too\n\n---\n\n## 📊 Firestore Collection\n\nRequired structure:\n```javascript\n{\n  id: \"notif-001\",\n  isRead: false,    // ← This field is queried\n  title: \"...\",\n  message: notifications.\n\n```javascript\nconst unsubscribe = await startNotificationBadge({\n  collectionName: 'notifications',  // Firestore collection\n  enableAppBadge: true,             // Update OS badge\n  autoInject: true                  // Auto-inject UI\n});\n\n// Later: stop listening\nunsubscribe();\n```\n\n### `injectNotificationLink(options)`\nManually create the badge element.\n\n```javascript\ninjectNotificationLink({ \n  navSelector: '.nav'\n});\n```\n\n### `stopNotificationBadge()`\nClean shutdown.\n\n```javascript\nwindow.NotificationBadge.stop();\n```\n\n### Events\n```javascript\nwindow.addEventListener('notificationCountChanged', (e) => {\n  console.log('New count:', e.detail.count);\n});\n```\n\n---\n\n## 🚀 Optional: Firebase Cloud Messaging
