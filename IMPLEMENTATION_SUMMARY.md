Architecture\n\n```\n┌──────────────────────┐\n│  Firestore           │\n│  notifications       │\n│  Collection          │\n│  (isRead: false)     │\n└──────────┬───────────┘\n           │\n           │ onSnapshot listener\n           │ (real-time)\n           │\n┌──────────v──────────────────┐\n│  notifications.js            │\n│  ├─ Query execution          │\n│  ├─ Count unread docs        │\n│  ├─ Update state             │\n│  └─ Dispatch UI updates      │\n└──────────┬──────────────────┘\n           │\n     ┌─────┴──────┐\n     │            │\n     v            v\n┌────────┐   ┌──────────┐\n│  DOM   │   │  App     │\n│ Badge  │   │  Badge   │\n│ Count  │   │  (PWA)   │\n└────────┘   └──────────┘\n```\n\n### Real-Time Flow\n\n1. **Initialization**\n   ```javascript\n   await startNotificationBadge({ collectionName: 'notifications' });\n   ```\n\n2. **Firestore Query**\n   ```javascript\n   const unreadQuery = query(\n     collection(db, 'notifications'),\n     where('isRead', '==', false)\n   );\n   ```\n\n3. **onSnapshot Listener**\n   ```javascript\n   onSnapshot(unreadQuery, (snapshot) => {\n     const count = snapshot.docs.length;\n     updateBadgeUI(count);\n   });\n   ```\n\n4. **UI Update**\n   - DOM: Badge displays count (e.g., \"5\")\n   - CSS: Badge shows/hides based on count\n   - PWA: OS app badge updates (if supported)\n\n---\n\n## Usage Examples\n\n### Basic Integration\n\n```html\n<script type=\"module\">\n  import { startNotificationBadge } from './notifications.js';\n  \n  // Auto-inject and start listening\n  await startNotificationBadge();\n</script>\n```\n\n### Advanced Control\n\n```javascript\n// Start with options\nconst unsubscribe = await startNotificationBadge({\n  collectionName: 'notifications',\n  enableAppBadge: true,\n  autoInject: true\n});\n\n// Listen for count changes\nwindow.addEventListener('notificationCountChanged', (e) => {\n  console.log('New count:', e.detail.count);\n});\n\n// Later: stop listening\nunsubscribe();\n```\n\n### Global API\n\n```javascript\n// Via window.NotificationBadge\nwindow.NotificationBadge.start();           // Start listening\nwindow.NotificationBadge.stop();            // Stop listening\nwindow.NotificationBadge.getCount();        // Get current count\nwindow.NotificationBadge.isListening();     // Check if active\n```\n\n---\n\n## Firestore Setup\n\n### Collection Structure\n\n**Collection Name:** `notifications`  \n**Required Field:** `isRead` (boolean)\n\n### Example Documents\n\n```javascript\n// Unread notification\n{\n  id: \"notif-001\",\n  isRead: false,              // ← This field is queried\n  title: \"Task Assigned\",\n  message: \"Review needed\",\n  createdAt: \"2026-06-09T10:00:00Z\"\n}\n\n// Read notification (won't be counted)\n{\n  id: \"notif-002\",\n  isRead: true,               // Won't match query\n  title: \"Task Completed\",\n  message: \"Done\",\n  createdAt: \"2026-06-09T09:00:00Z\"\n}\n```\n\n### Marking as Read\n\n```javascript\nimport { doc, updateDoc } from 'firebase/firestore';\n\nawait updateDoc(doc(db, 'notifications', 'notif-001'), {\n  isRead: true\n});\n\n// Badge count decreases automatically\n```\n\n---\n\n## Deployment Steps\n\n### 1.
