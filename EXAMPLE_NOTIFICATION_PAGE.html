🚀 Quick Start

\n

1. Minimal Setup

\n 
import { startNotificationBadge } from './notifications.js';\n\n// Auto-inject and start listening\nawait startNotificationBadge();
\n\n

2. Custom Collection

\n 
await startNotificationBadge({\n  collectionName: 'alerts',\n  enableAppBadge: true\n});
\n\n

3. Manual Control

\n 
// Get current count\nconsole.log(window.NotificationBadge.getCount());\n\n// Check if listening\nconsole.log(window.NotificationBadge.isListening());\n\n// Stop listening\nwindow.NotificationBadge.stop();
\n

📖 API Reference

\n\n

startNotificationBadge(options)

\n

Start listening to unread notifications.

\n 
const unsubscribe = await startNotificationBadge({\n  collectionName: 'notifications',  // Optional\n  enableAppBadge: true,              // Optional\n  autoInject: true                   // Optional\n});
\n\n

injectNotificationLink(options)

\n

Manually create the notification UI element.

\n 
injectNotificationLink({ \n  navSelector: '.nav' \n});
\n\n

stopNotificationBadge()

\n

Stop listening and reset badge.

\n 
window.NotificationBadge.stop();
\n\n

getNotificationCount()

\n

Get current unread count.

\n 
const count = window.NotificationBadge.getCount();

📋 Firestore Document Structure

\n

Required field: isRead (boolean)

\n 
{\n  id: \"notif-001\",\n  isRead: false,           // ← Queried field\n  title: \"Task Assigned\",\n  message: \"Review needed\",\n  createdAt: \"2026-06-09T10:00:00Z\",\n  userId: \"user-123\"\n}
\n
\n Note: Any collection with an isRead field can be used.\n Not limited to \"notifications\" collection.\n

🐛 Troubleshooting

\n \n

Badge Not Showing?

\n
    \n
  • Check CSS is loaded: css/style.css
    • \n
  • Verify .nav element exists
    • \n
  • Open DevTools Console for errors
    • \n
\n\n

Count Not Updating?

\n
    \n
  • Verify Firestore collection exists
    • \n
  • Ensure documents have isRead: false field
    • \n
  • Check Firestore security rules allow reads
    • \n
  • Run: console.log(window.NotificationBadge.isListening())
    • \n
\n\n

Firebase Not Initialized?

\n
    \n
  • Verify firebase-config.js is imported
    • \n
  • Check Firebase credentials
    • \n
  • Ensure network connection is working
